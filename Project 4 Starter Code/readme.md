# 🏠 Real Estate Platform

## 📌 Description

Real Estate is a full-stack web application for renting and managing properties.
It allows users to browse listings, view detailed property information, and securely log in to manage their accounts and properties.

---

## 🚀 Features

* 🔐 User Authentication (Login & Register)
* 🏡 Browse available properties
* 🔍 Search and filter properties
* 📄 View detailed property information
* ➕ Add new properties
* ✏️ Edit and delete listings
* 🗄️ Database integration for storing users and properties

---
